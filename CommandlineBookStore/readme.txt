 BookStore System
This C++ application simulates a simple bookstore management system.
The program allows users to manage a bookstore's inventory, add books to a shopping cart, and proceed to checkout.
The application provides a command-line interface for interacting with the bookstore and cart.

How to Run
1. Clone or download the source code to your local machine.
2. Compile the source code using a C++ compiler. Example with g++(dev++,visual studio code).
3. Run the program.
4. Follow the on-screen instructions to interact with the bookstore system.

Instructions
1. Viewing Books: Select option 1 to display all books currently in the inventory.
2. Adding a Book to Cart: Select option 2 and enter the ISBN of the book you wish to add to your cart.
3. Removing a Book from Cart: Select option 3 and enter the ISBN of the book you wish to remove from your cart.
4. Checkout: Select option 4 to proceed with checkout. The inventory will be updated based on the books in your cart.
5. Adding a New Book to Inventory: Select option 5 and follow the prompts to add a new book to the inventory. Ensure that the ISBN is unique.
6. Exiting the Program: Select option 6 to exit the program.

Error Handling
 The program includes error checking for user inputs.
 Such as validating menu choices and ensuring valid data types for book details.
 When adding books to the inventory.
 The program checks for duplicate ISBNs to prevent adding the same book multiple times.
